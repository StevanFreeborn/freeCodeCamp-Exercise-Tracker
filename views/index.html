Exercise Tracker

Documentation

Welcome to the Exercise Tracker service. This documentation should supply you with all the information you need to start making requests. It is a really simple api service built to complete one of the projects for the freeCodeCamp Back End Development and APIs certification.

Base Url
https://freecodecampexercisetracker.herokuapp.com
Create User Endpoint

An endpoint that will allow you to pass a username value and create a user which will be returned as a JSON object.

/api/users
Example Request:
POST https://freecodecampexercisetracker.herokuapp.com/api/users

Content-Type:
application/x-www-form-urlencoded

username:
The desired username for the user.
Example Response:
{
 "username": "stevan.freeborn",
 "_id": "6275425ad2fe0db53499fb59",
 "__v": 0,
}
Get Users Endpoint

An endpoint that will return all the users.

/api/users
Example Request:
GET https://freecodecampexercisetracker.herokuapp.com/api/users
Example Response:
[
 {
  "username": "stevan.freeborn",
  "_id": "6275425ad2fe0db53499fb59",
  "__v": 0,
 },
 {
  "username": "stevan",
  "_id": "7275425ad2fe0db53499fb59",
  "__v": 0,
 }
]
Log Exercise Endpoint

An endpoint that will allow you to log exercise for a user.

/api/users/{id}/exercises
Example Request:
POST https://freecodecampexercisetracker.herokuapp.com/api/users/7275425ad2fe0db53499fb59/exercises

Content-Type:
application/x-www-form-urlencoded

description:
A description of the exercise being logged.

duration:
The length of the exercise being logged in minutes.

date:
The date on which the exercise was performed. Suggested format YYYY-MM-DD.
Example Response:
{
 "username": "stevan.freeborn",
 "description": "treadmill",
 "duration": 5,
 "date": "Mon Feb 01 1993",
 "_id": "6275425ad2fe0db53499fb59",
}
Get Exercise Logs Exercise Endpoint

An endpoint that will allow you to get the exercise logs for a user. Can also filter the log based on date range and limit the number of logs returned.

/api/users/{id}/logs
Example Request:
GET https://freecodecampexercisetracker.herokuapp.com/api/users/7275425ad2fe0db53499fb59/logs
Example Response:
{
 "username": "stevan.freeborn",
 "count": 2,
 "_id": "6275425ad2fe0db53499fb59",
 "date": [
   {
     "description": "test",
     "duration": 6,
     "date": "Sat May 07 2022"
   }
 ]
}
